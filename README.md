# code_input

Code input widget : segmented code input control


## install it

```
dependencies:
  code_input: ^0.0.2
```

## import it

```
import 'package:code_input/code_input.dart';
```

## Example

```
return Scaffold(
  appBar: AppBar(
    title: Text("Code Input"),
  ),
  body: Center(
    child: CodeInput(
      //how many input widget you want  
      lenght: 5,
    ),
  ),
);
```
